# european-red-list-dashboard
An interactive analysis of biodiversity risk and conservation status across European species.

## Links
- **Live App:** [Click here to view the dashboard](https://ovin-european-red-list-dashboard-2024.streamlit.app/)
- **Dataset Source:** [European Red List 2024 — European Environment Agency](https://sdi.eea.europa.eu/data/9a752c28-cb5f-4ead-9922-2a8173e0306b)

## Dashboard Features
- Interactive filters by Species Group, Conservation Status, Population Trend and Endemic status
- Toggle to include/exclude Unknown classifications
- 4 KPI metrics — Total Species, Threatened Species, % Threatened, Endemic Species
- Bar chart, Treemap, Heatmap, Horizontal bar chart, Donut chart
- Interactive species explorer data table

## Technologies Used
- Python, Streamlit, Pandas, Plotly

## How to Run Locally
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
